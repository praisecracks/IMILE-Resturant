# ÌMÍLÈ • Afro-Futurist Gastronomy

![ÌMÍLÈ Preview](https://images.unsplash.com/photo-1514362545857-3bc16c4c7d1b?auto=format&fit=crop&q=80&w=1200)

**ÌMÍLÈ** (meaning *Heritage* or *Root* in a futuristic context) is an ultra-premium, immersive dining experience designed for the elite Skyline of Neo-Lagos (2042). This project is a masterclass in cinematic web design, combining high-end hospitality with Afro-Futuristic aesthetics.

## 🌌 The Experience

The website is designed to evoke a sense of "expensive minimalism." It moves away from traditional restaurant layouts, favoring a storytelling approach where every scroll feels like a transition between different layers of a Michelin-star evening.

### Key Features
- **Cinematic Storytelling**: Uses GSAP ScrollTrigger to reveal content with parallax and text masking effects.
- **Afro-Futurist Visuals**: A custom "Deep Espresso" and "Bronze Gold" theme infused with subtle geometric African patterns.
- **Horizontal Visual Journey**: A pinned gallery section that scrolls horizontally to showcase the "Eko Skyline" atmosphere.
- **Smooth Navigation**: Integrated Lenis smooth-scrolling for a buttery-smooth desktop feel.
- **Interactive Menu**: A high-touch menu showcase featuring elevated Nigerian classics like *Gold Leaf Jollof* and *Lobster Moin-Moin*.
- **Reservation Engine**: A custom-designed, glassmorphic booking interface tailored for an exclusive concierge experience.

## 🛠️ Technological Foundation

This application is built with a modern, production-grade tech stack prioritizing performance and fluidity.

### Core Stack
- **Framework**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4.0](https://tailwindcss.com/)
- **Animations**: [GSAP](https://gsap.com/) & [Motion (Framer Motion)](https://motion.dev/)
- **Smooth Scroll**: [Lenis](https://lenis.darkroom.engineering/)
- **Icons**: [Lucide React](https://lucide.dev/)

### Design tokens
- **Typography**: `Inter` (Sans), `Playfair Display` (Serif/Italic), `Outfit` (Display)
- **Primary Color**: `#D98C45` (Bronze Gold)
- **Background**: `#0A0705` (Deep Espresso)

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/praisecracks/IMILE-Resturant.git
   cd imile-gastronomy
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
